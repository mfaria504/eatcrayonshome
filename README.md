# EatCrayons — Business Growth Partners

Static HTML replica of [eatcrayons.com](https://eatcrayons.com) — a single-file, self-contained website with no build step required.

## Stack

- Pure HTML / CSS / JavaScript — zero dependencies
- Google Fonts (Open Sans)
- Fully responsive (mobile, tablet, desktop)
- Accessible (semantic HTML, ARIA labels, `prefers-reduced-motion` support)

## Sections

1. **Announcement Bar** — scrolling marquee ticker
2. **Header** — sticky nav with scroll-blur effect
3. **Hero** — animated cycling headline, Venn diagram
4. **About** — founder pain-point narrative
5. **Services** — Strategy, Infrastructure, Delivery cards
6. **Trust** — social proof stats ($175M, 92%, 320+)
7. **Testimonials** — client quote cards
8. **Pricing** — Growth Foundation & Growth Engine plans
9. **CTA** — contact section
10. **Footer + Watermark**

## Usage

Open `index.html` directly in any browser — no server required.
